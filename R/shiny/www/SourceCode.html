The data parsing was done in a combination of Node.js and Python. The Node.js scraper was written by Nigel Ticknor, and the Python one was written by Paul DeMange. We did this to minimize the time it took to scrape the data by optimizing our respective scrapers to our half of the data.

The R code was written by Nigel Ticknor, Shaun Boerner, and Paul DeMange. Nigel wrote code to import and clean the data, Paul wrote some supporting functions, and all three of us wrote code to visualize the data. Nigel also wrote code to build and deploy a Shiny Dashboard.

Nigel also wrote the additional JavaScript code that powers the DataBreach Dashboard alongside the Bootstrap framework.
